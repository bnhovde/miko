## Miko

![Image showing editor](https://bardhovde.com/assets/miko.png)

Simple pixel editor with frame animation. Share your creations via an URL.

Blog post: [https://bardhovde.com/posts/miko/](https://bardhovde.com/posts/miko/)

```bash
npm install
npm run dev
```
